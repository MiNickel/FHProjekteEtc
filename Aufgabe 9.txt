Teil 1:

Ändern Sie den Vornamen von Herrn Paulus (siehe Vorlesung!) und den Nachnamen
von Frau Godebracht-Paulus (die haben geheiratet, siehe Vorlesung!)

Lösung:

UPDATE angestellter
SET name = 'Godebracht-Paulus'
WHERE name = 'Godebracht' AND vorname = 'Marion'

UPDATE angestellter
SET vorname = 'Ottmar'
WHERE name = 'Paulus' AND vorname = 'Otmar'
